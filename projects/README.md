# Projects — Frank Developments

One folder per Frank project currently being promoted (10+). Each folder is the **fuel** for NUNU's work
on that project — the more real material it holds, the less generic NUNU can possibly be. NUNU never drafts
for a project without reading its folder first.

## How to add a project (Yana)
1. Copy the **`_TEMPLATE`** folder and rename it to the project (e.g. `projects/the-newmarket/`).
2. Drop in that project's materials (see the template's README for exactly what).
3. Tell NUNU the project's ready — it onboards it and can start producing.

## What each project folder holds
- `guidelines.md` (or the source docs) — positioning, do/don't, mandatories, legal/compliance lines.
- `brand-voice.md` — this project's tone + voice notes.
- `design/` — the **exported Claude Design project archive** (colours, type, imagery, logo lockups, hero).
- `briefs/` — campaign briefs (goal, audience, project moment, dates/prices).
- `campaigns/` — where NUNU saves finished EDMs (`.html`) + social/activation sets.
