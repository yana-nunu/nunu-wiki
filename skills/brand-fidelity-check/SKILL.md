# brand-fidelity-check

The checklist that enforces NUNU's Critical Rule. Run it on every EDM, caption, story idea, or piece of
copy before handoff. A single failed item = not done; rebuild from source. Be adversarial with yourself —
the goal is to *catch* generic before Yana (or a buyer) does.

## The one question that governs everything
> **Could this belong to any other developer, any other project?**
If yes, it's generic — rebuild until it could only be THIS project.

## The checklist (all must pass)
1. **Sourced, not invented.** Every claim, feature, price, date, and detail traces to the project's guidelines / brief / design reference. Nothing fabricated or assumed. (Inferences, if any, are flagged to Yana, not smuggled in.)
2. **Ownably specific.** The headline + hero could not be lifted onto another project. Real locale, real amenity, real buyer, real moment — not "stunning lifestyle" abstractions.
3. **Voice is Frank's.** Matches the brand voice notes + this project's tone. Structured storytelling, not brochure-speak.
4. **Superlative purge.** No "stunning / luxurious / nestled / iconic / elevated / bespoke / unparalleled" unless the guidelines explicitly earn that word. Concrete detail beats adjective every time.
5. **Guideline mandatories present.** Every required disclaimer, legal/REA line, logo lockup, and do/don't from the project guidelines is honoured.
6. **No AI tells.** No hollow tricolons ("more X, more Y, more Z"), no "Imagine a place where…", no "Nestled in the heart of", no em-dash-itis, no empty parallelism, no promise-without-substance. If a sentence could be in a thousand emails, cut or ground it.
7. **The details are right.** Dates, prices, addresses, project name spelling, CTA link — all correct. A wrong fact in a buyer email is a real failure, not a typo.
8. **Renders + sends** (for EDMs): passes email-client + dark-mode test, HubSpot-importable, images have alt text, unsubscribe + address present.
9. **Yana can own it.** It reads like something she'd be proud to send under her name — because she's the final voice and it lands at the department-lead standard she's gunning for.

## On failure
Don't polish generic into less-generic. Go back to the project's materials and rebuild the failing part
from the real source. Generic is not a finish state.
