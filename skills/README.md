# NUNU skill library

NUNU's procedural memory — the playbooks it pulls from to do Yana's marketing work at Frank Developments.
Every skill serves the one rule: **source-grounded, never generic** (see [[../soul]]).

Standalone — these are NUNU's own, developed independently of MOMO.

## Skills
- [[edm-campaign/SKILL|edm-campaign]] — produce a Frank-grade EDM as email-safe, HubSpot-ready HTML.
- [[email-html-scaffold/SKILL|email-html-scaffold]] — the reusable email-safe HTML template the EDM fills.

- [[community-engagement/SKILL|community-engagement]] — Instagram stories, community/DM replies, events + activations.

## Contract
Each skill is a folder with `SKILL.md` (frontmatter `name` + `description`, then When-to-use / Inputs /
Method / Output / Gotchas). Build only from proven method; append gotchas as NUNU learns them; keep it tight.
